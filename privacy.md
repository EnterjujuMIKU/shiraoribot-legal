# Privacy Policy: ShiraoriBot
---

Dernière mise à jour : 2025-11-18

# 1. Introduction

Cette Politique de Confidentialité (« Privacy Policy ») explique quelles données ShiraoriBot collecte, comment elles sont utilisées, stockées et protégées.
En utilisant le bot, vous acceptez les pratiques décrites ci-dessous.

ShiraoriBot respecte les principes du RGPD (Règlement Général sur la Protection des Données) dans la mesure applicable aux bots Discord.

# 2. Données collectées automatiquement

ShiraoriBot collecte uniquement les informations nécessaires au fonctionnement de ses fonctionnalités.
Ces données proviennent de l’API Discord.

## 2.1 Identifiants Discord

Le bot peut collecter ou manipuler :
• User ID
• Server ID
• Channel ID
• Message ID
• Role ID (si une fonctionnalité l’exige)

Ces identifiants ne contiennent aucune donnée personnelle directe, mais ils permettent d’interagir avec Discord.

## 2.2 Contenu des commandes

Lorsque vous exécutez une commande du bot :

Le bot traite :
• le contenu du message ou slash-command,
• les arguments envoyés,
• certaines données d'état (ex. progression d'une quête, statistiques locales, paramètres).

Le bot ne lit pas les messages ordinaires qui ne lui sont pas adressés.

## 2.3 Données de logs internes

Pour assurer un fonctionnement stable, le bot peut temporairement enregistrer :
• des erreurs techniques,
• des événements système limités (déconnexions, crashs, redémarrages).

Ces logs ne contiennent pas vos conversations.

# 3. Données non collectées

ShiraoriBot ne collecte pas :
• adresses IP,
• emails, numéros ou informations de compte externes,
• données sensibles (santé, politique, orientation, etc.),
• pièces jointes, images, fichiers,
• messages privés entre utilisateurs,
• contenu des messages non liés au bot.

Le bot n’analyse, ne scanne et ne lit pas vos conversations.

# 4. Finalité du traitement

Les données collectées servent exclusivement à :

## 4.1 Fonctionnement

• exécuter les commandes du bot,
• permettre la progression dans les systèmes de quêtes, d'expérience, de mini-jeux, etc.,
• offrir des outils de modération légère,
• stocker des paramètres configurés par les administrateurs.

## 4.2 Sécurité et stabilité

• prévenir les abus,
• assurer la maintenance technique,
• analyser les erreurs afin d’améliorer la stabilité du bot.

Le bot n'utilise pas vos données à des fins commerciales ou publicitaires.

# 5. Stockage des données
## 5.1 Lieu et durée

Les données éventuellement stockées le sont :
• uniquement si nécessaire au fonctionnement du bot,
• uniquement dans une base de données gérée par le Développeur,
• supprimées lorsqu’elles ne sont plus nécessaires (ex. réinitialisation, désinstallation du bot, purge manuelle).

## 5.2 Sécurité

Des mesures raisonnables de protection sont mises en place :
• limitation d'accès,
• stockage non public,
• utilisation de pratiques de sécurité courantes.

Aucune sécurité n’est infaillible, mais tout est fait pour réduire les risques.

# 6. Partage des données

ShiraoriBot ne partage ni ne vend aucune donnée à des tiers.
Aucun partenaire, tiers ou service externe n'a accès aux données du bot.

Les seules personnes pouvant consulter les données sont :
• le Développeur,
• éventuellement un administrateur technique autorisé si cela devient nécessaire.

Aucune donnée n’est transmise en dehors de l’Union européenne, sauf si le service d’hébergement du bot le nécessite (ex. serveurs Discord situés mondialement).

# 7. Droits des utilisateurs (RGPD)

Selon le cadre du RGPD, vous disposez de droits concernant vos données :

## 7.1 Droit d’accès

Vous pouvez demander quelles données liées à votre User ID sont stockées.

## 7.2 Droit de rectification

Vous pouvez demander la correction d’informations incorrectes si le bot en stocke.

## 7.3 Droit à l’effacement

Vous pouvez demander la suppression complète de vos données, sous réserve des données nécessaires à la sécurité.

## 7.4 Droit à la limitation ou opposition

Vous pouvez refuser l'utilisation de vos données en cessant d'utiliser le bot et en demandant la suppression de vos données.

Toutes ces demandes peuvent être adressées directement au Développeur.

# 8. Suppression des données

L’Utilisateur ou l’administrateur du serveur peut demander :
• suppression d'un compte de progression,
• réinitialisation de données,
• retrait du bot d’un serveur (supprime la collecte future),
• suppression totale des données associées à un User ID.

Une demande simple via Discord suffit.

# 9. Mineurs

ShiraoriBot ne cible pas spécifiquement les mineurs, mais peut être utilisé par eux conformément aux règles Discord (âge minimum 13 ans ou âge légal local en france).
Aucune donnée sensible ou profilage n’est effectué.

# 10. Modifications de la politique

La présente Politique peut être mise à jour à tout moment.
Les modifications seront affichées sur la page officielle.
La poursuite d’utilisation du bot vaut acceptation.

# 11. Contact

Pour toute question, demande d’accès ou suppression de données, vous pouvez contacter le Développeur directement ou dans le server communautaire officiel via Discord.
