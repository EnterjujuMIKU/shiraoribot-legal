# Privacy Policy – ShiraoriBot
Dernière mise à jour : 2025-11-18

## 1. Données collectées
ShiraoriBot peut collecter :
- Identifiants Discord (User ID, Server ID, Message ID) ;
- Messages envoyés via les commandes du bot ;
- Paramètres configurés par les administrateurs.

Ne sont pas collectés :
- adresses IP ;
- messages privés hors commandes ;
- fichiers, images ou données sensibles.

## 2. Utilisation des données
Les données servent uniquement à :
- exécuter les commandes du bot ;
- gérer les systèmes de progression (quest, stats…) ;
- assurer la modération légère.

## 3. Stockage
Les données sont conservées uniquement si nécessaires au fonctionnement. Elles peuvent être supprimées lorsqu’elles ne sont plus utiles.

## 4. Partage
Aucun partage ou vente de données. Elles sont utilisées uniquement par le développeur.

## 5. Sécurité
Des mesures raisonnables sont appliquées pour protéger les données, sans garantie absolue.

## 6. Suppression des données
Vous pouvez demander la suppression de vos données en contactant le développeur.

## 7. Modifications
Cette politique peut être mise à jour. L’utilisation continue du bot vaut acceptation des changements.
